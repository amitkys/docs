## Installation


```bash tab="npm"
npx shadcn@latest add @sabnamui/loader
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add @sabnamui/loader
```

```bash tab="yarn"
yarn shadcn@latest add @sabnamui/loader
```

```bash tab="bun"
bunx --bun shadcn@latest add @sabnamui/loader
```
