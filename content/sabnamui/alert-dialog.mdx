## Installation

```bash tab="npm"
npx shadcn@latest add https://maybedocs.id0.uk/ui/alert-dialog.json 
```

```bash tab="pnpm"
pnpm dlx shadcn@latest add https://maybedocs.id0.uk/ui/alert-dialog.json
```

```bash tab="yarn"
yarn shadcn@latest add https://maybedocs.id0.uk/ui/alert-dialog.json
```

```bash tab="bun"
bunx --bun shadcn@latest add https://maybedocs.id0.uk/ui/alert-dialog.json
```
